# 07 — Cannes Strategy

Neptura's creative platform and execution roadmap for Cannes Lions.

## The Idea

> **"The Earth's First Audit."**

The earth has been sequestering carbon, filtering water, and regenerating ecosystems for 3.8 billion years — without a single auditor. Neptura is the first entity capable of reading the planet's books. Every execution derives from this single, irrefutable insight.

## The Standard

Work is approved for Cannes entry only if it passes three questions:
1. Would this embarrass us next to a Grand Prix winner?
2. Could any competitor claim this idea?
3. Is the execution as precise as the data we sell?

If any answer is no — kill it.

No green leaves. No smiling children in fields. No "sustainable" without a number. No work a competitor could have made.
