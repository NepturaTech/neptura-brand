## [0.1.0] — 2026-01-21

### Added
- Initial Brand OS — strategy, identity, and guidelines.
- Logo system: icon mark + wordmark in 6 color variants (SVG + PDF).
- Color tokens: `neptura-colors.json` + `neptura-colors.css`.
- Typography specification: `neptura-typography.json`.
- Brand strategy docs: foundations, decision framework, messaging matrix.
- Visual identity docs: territories, style directions, logo routes, UI components.
- Brand guidelines MVP — quick execution rules.
- Dashboard specification — Mayor's Cockpit + Exporter's Audit.
- Packaging framework — Smart Tag + Certificate Asset.
- Go-to-market plan — "Operation Andes" (3 phases, 18 months).
- Design agent instructions for execution.
