## The 5 Non-Negotiables

1. Logo proportions are never altered.
2. Signal Green is used only for success/key data — never as a full background.
3. Monospace fonts are for data only — never for long-form reading.
4. Photography shows dignity and technology — never poverty or generic "rural" imagery.
5. Use the approved terminology (see [messaging-system.md](../01-strategy/messaging-system.md)).
