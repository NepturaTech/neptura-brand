# Visual Territories
## Definición del Territorio Visual: "Deep Data Nature"

### Paleta Cromática

| Token | Hex | Significado |
|---|---|---|
| Neptura Deep Blue | `#0A1E26` | Profundidad, seriedad, monitoreo satelital |
| Signal Green | `#00FF94` | Datos positivos, zonas óptimas, botones CTA |
| Alert Orange | `#FF6B00` | Zonas de riesgo, intervención urgente |
| Neutros | Grises fríos | Fondos secundarios, texto secundario |

> **Note:** The production palette uses slightly different values. See [colors/neptura-colors.json](./colors/neptura-colors.json) for the canonical tokens.

---

### Tipografía

- **Titulares:** Sans Serif Grotesca, sólida y geométrica — Inter o Space Grotesk.
- **Data/UI:** Monospace — JetBrains Mono. Usado SOLO para números, coordenadas GPS, y valores financieros. Transmite rigor y precisión.

---

### Tratamiento Fotográfico

- Fotos satelitales reales intervenidas con capas de UI (líneas de nivel, perímetros vectoriales).
- Retratos de productores mirando a cámara: dignidad, sin filtros "nostálgicos", iluminación nítida.
- **Prohibido:** Fotos de pobreza rural o escenas "campesinas" genéricas.
